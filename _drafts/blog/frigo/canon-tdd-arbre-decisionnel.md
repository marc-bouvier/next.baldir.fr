# Arbre décisionnel Canon TDD


TDD ça a l'air simple mais il y a plein de pièges dans lesquels on tombe facilement quand on débute.

Avec Tidy First, Kent Beck propose "Canon TDD" : un déroulé un peu plus détaillé qui permet de mieux construire son workflow TDD.

Très pratique pour commencer ! Notez la présence d'une étape d'écrire une liste de scénario de test qu'on souhaite couvrir.

Kent Beck dans Tidy First : [https://tidyfirst.substack.com/p/canon-tdd](https://tidyfirst.substack.com/p/canon-tdd)
