Delete a word : `daw`

Delete a word (including dashes) : `daW`

Run a command : `:!<command>` ex. `:!ls`

Run a command and get the result in the current file :  `:%!<command>` `:%!ls`

Fold : Command + ↓

Unfold : Command + ↑

Completion de path : Ctrl + x + f
